   Le démineur de l'internet  
|-----------------------------------|

Pour accéder au site internet , vous devez :

- Ouvrir le fichier intitulé "Page_accueil.html"

- Cliquer sur une des trois difficulté proposé par le site (ou l'easter egg si vous le trouvez)

Et puis vous pouvez revenir au menu depuis le bouton Accueil



IMPORTANT: 
	Le site est fait principalement fait pour fonctionner sur Google Chrome et Mozilla Firefox , il peut potentiellement fonctionner sur d'autres navigateurs mais il n'est pas prévu pour cela et il est donc possible que des bugs se produisent ou certains éléments ne s'affichent pas comme voulu.


MODE DEBUG :

	Pour activer le mode debug (uniquement accessible pour les professeurs et les développeurs du projet) vous devez cliquer sur l'image invisible qui se situe juste au dessus du bouton d'accueil et rentrer le mot de passe "Ordipass" pour activer le mode debug, il vous affichera dans la console du site (que vous pouvez ouvrir en appuyant sur la touche f12 de votre clavier) le tableau avec toutes les bombes et leurs positions ainsi que les numéros autours des bombes (les bombes étant représenté par le mot 'true'. Pour plus de lisibilité , ouvrez la console avant d'activer le mode debug.
